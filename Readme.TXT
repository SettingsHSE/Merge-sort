Сортировка слиянием (англ. merge sort) — алгоритм сортировки, демонстрирующий пример использования принципа «разделяй и властвуй». 
Сначала задача разбивается на несколько подзадач меньшего размера. Затем эти задачи решаются с помощью рекурсивного вызова или непосредственно, если их размер достаточно мал. Наконец, их решения комбинируются, и получается решение исходной задачи.

Формат ввода
На вход программа принимает 2 строки: 
в первой строке целое число N - количество элементов для сортировки. 
Гарантируется, что 1 <= N <= 1000000. 
во второй строке N целых чисел через пробел. 
Обозначим каждое из N входящих чисел как P, тогда гарантируется, что 0 <= P <= 1000000. 
В первой строке — количество элементов во входном массиве. 
Во второй строке — массив с целыми элементами (со знаком) через пробел.

Формат вывода
На выход программа должна выводить 1 строку: 
в первой строке M целых чисел через пробел. 
Все числа должны быть по неубыванию, могут быть повторы, и числа должны быть из первоначального набора. 
Предполагается, что 1 <= M <= 1000000, и точно соответствует входящему N.
